C class

Dropdown extends Floater

Assign a floater panel to a {@link https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent MouseEvent}

Constructor

new Dropdown(options)
options : Object optional
mouseevent : string optional

MouseEvent that toggles floater

removeOnBlur : boolean optional

Hide dropdown when user clicks outside or focuses elsewhere

timeout : number optional

Milliseconds to wait before hiding dropdown after mouseleave

Methods

MethodParametersReturnDescription
shownonethisShows the dropdown and adds the -active class to the anchor element
hidenonethisHides the dropdown and removes the -active class from the anchor element
toggle[shouldHide:boolean]thisToggles the dropdown visibility. If shouldHide is provided, will explicitly show or hide

Examples

<caption>JS</caption>
new Dropdown({
    content: createElement({
        class: 'bg-white pad',
        content: 'Hello World'
    }),
    anchor: '#hi-button'
})
<caption>HTML</caption>
<komp-dropdown anchor="#hi-button">
    Hello World
</komp-dropdown>
