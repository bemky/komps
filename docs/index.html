Komps

Komps is a library of style-less javascript components and elements for building a UI. Komps is platform-agnostic, with each component is fully tree-shakeable by an bundler.

Installation

npm install komps

Usage

Import only the methods you use. 

import {Modal} from 'komps';

new Modal({content: "Hello World"});
